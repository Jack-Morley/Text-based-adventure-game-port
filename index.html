HOW TO PLAY: Can only move either north, east, south or west. The direction that you can move is indicated by the box on the top. Some rooms contain treasure to collect, however it is not mandatory to collect all the coins. Some rooms contain monsters which will attack, dealing damage to you. If your HP reaches 0 you lose and must try again. In the box directly above, you are able to see the descriptions of the rooms that you are in. The (?) symbol indicates that there is something you can interact with. Interactions work only once per room. Find the exit to win!
